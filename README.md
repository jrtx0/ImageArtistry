# ImgArtistry
Cartoonise and Handpainted your image.

### Usage
* Get source code from Github:
```
  https://github.com/jrtx0/ImgArtistry.git
```
* Import project into Pycharm or directly run **main.py** in terminal:
```
  python3 main.py
```
